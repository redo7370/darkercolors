## 📦 Installation

1. Open Visual Studio Code
2. Go to Extensions (`Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Search for "Super Dark Colors" 
4. Click Install
5. Go to Settings (`Ctrl+,` / `Cmd+,`)
6. Search for "Color Theme"
7. Select your preferred variant:
